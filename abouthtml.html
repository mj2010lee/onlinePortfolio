What is HTML, the Basic Language of the Web?

Post written by Myungjae Lee on January 6th, 2025

There are three main languages of the web, HTML, CSS, and JavaScript. All modern websites and web-apps are built using these. In this post, we will be focusing on HTML. We will learn what HTML is all about, and why you too, should learn it.

What exactly is HTML?

HTML stands for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage. Please note that HTML is not a programming language.

Timothy J. Berners-Lee first suggeted HTML in 1990 as a markup language for a new technology, the World Wide Web(a.k.a. WWW). Over time, HTML evolved into what became the core backbone of the modern internet. HTML consists of elements describing contents that the web can display: title, text, paragraphs, images, video, and a lot more. HTML 'marks up' website, thus the name 'Markup Language'.

An example of an HTML code looks like this : <p>Hello, World!</p>

  1. <p> is an opening tag. The 'p' here means paragraph.
  2. Hello, World! is the actual element displayed to the screen.
  3. </p> is a closing tag. It means the code is finished.

This code displays the following to the screen: Hello, World!

Why should we learn HTML?

Many people learn HTML for various reasons, and here are some of them:

  • To be able to use the fundamental web language
  • To make beautiful websites without relying on tools like Wordpress, or AIs like chatGPT
  • To impress friends
  • To build our own web apps
  • To have fun!!!
